    Check domain security settings including SPF, DMARC, DKIM, BIMI, and more.

    == Description ==

    Domain Security Checker allows you to verify various security settings for any domain, including:

    * SPF (Sender Policy Framework)
    * DMARC (Domain-based Message Authentication)
    * DKIM (DomainKeys Identified Mail)
    * BIMI (Brand Indicators for Message Identification)
    * DNSSEC
    * TLS Support
    * MTA-STS
    * Zone Transfer settings

    Features:

    * Easy to use interface
    * Detailed security analysis
    * Overall security score
    * Strengths and improvements summary
    * Security risk assessment
    * Multilingual support (English and Dutch)
    * Widget and shortcode support

    == Installation ==

    1. Upload the 'domain-checker' folder to the '/wp-content/plugins/' directory
    2. Activate the plugin through the 'Plugins' menu in WordPress
    3. Use the shortcode [domain_checker] in any post or page
    4. Optionally, add the Domain Security Checker widget to your sidebar

    == Usage ==

    Shortcode: [domain_checker]
    Widget: Add the "Domain Security Checker" widget to any widget area
